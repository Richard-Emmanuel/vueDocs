---
title: TS with Composition API
description: A guide to using TypeScript with the Composition API in Vue.js.
---

In Vue.js, the Composition API is a powerful way to organize and reuse logic in your components. This guide will cover the basics of using TypeScript with the Composition API, including how to define props and emits, how to use reactive and ref, and how to create composables.
It will also provide examples to help you understand how to use TypeScript effectively with the Composition API in your Vue.js applications.
